# Graphing Calculator

A feature-rich graphing calculator application built with Python and tkinter.

## Features

- **Graph Functions** - Plot mathematical expressions with customizable axis ranges
- **Visual Graphs** - Interactive matplotlib graphs with grid and axes
- **Quick Calculator** - Calculate mathematical expressions instantly
- **Supported Functions** - sin, cos, tan, sqrt, log, exp, and more

## Installation & Usage

### Option 1: Run as Python Script (Recommended for Development)

1. **Install Python 3.7+** from [python.org](https://www.python.org/downloads/)

2. **Install dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the app:**
   ```bash
   python test.py
   ```

### Option 2: Run as Windows Executable (No Python Required)

1. **Install PyInstaller:**
   ```bash
   pip install pyinstaller
   ```

2. **Create the .exe:**
   ```bash
   pyinstaller --onefile --windowed test.py
   ```

3. **Run the app:**
   - Find `test.exe` in the `dist` folder
   - Double-click to launch

## Example Functions

Try plotting these:
- `x**2` - Parabola
- `sin(x)` - Sine wave
- `1/x` - Hyperbola
- `sqrt(x)` - Square root
- `exp(x)` - Exponential (use range -2 to 2)
- `x**3 - 2*x` - Cubic function

## Supported Mathematical Operations

- **Trigonometric:** sin, cos, tan, asin, acos, atan
- **Hyperbolic:** sinh, cosh, tanh
- **Logarithmic:** log (natural), log10
- **Other:** sqrt, exp, abs, ceil, floor
- **Constants:** pi, e
- **Operators:** +, -, *, /, ** (power)

## Quick Calculator Section

Use the "Quick Calculator" to evaluate single expressions:
- `2 + 2`
- `sqrt(16)`
- `sin(pi/2)`
- `2**10`

## Requirements

- Python 3.7 or higher
- numpy
- matplotlib
